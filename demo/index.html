popupable

Integrated popup images that expand from the page, with gallery, thumbnails, and zoom support.

Use data-popupable attributes directly in your markup.

Features

  • Expands from the source element for an integrated transition
  • Grouped galleries with keyboard, wheel, and swipe navigation
  • Optional counter, captions, and thumbnail strip
  • Pinch and pan zoom on touch devices
  • No-upscale mode to cap display at native image size

Basic Usage

Add data-popupable to any image.
